## How to Take Screenshots

### Android:
```bash
flutter run
# App run karo, phir screenshot lein
```

### iOS Simulator:
1. App run karo iOS simulator mein
2. Cmd + S press karo screenshot ke liye
3. Screenshots Desktop par save honge

### Physical Device:
- Android: Power + Volume Down
- iOS: Power + Home (ya Side + Volume Up)

## Screenshot Guidelines

- **Size**: Minimum 1080x1920 (portrait) ya 1920x1080 (landscape)
- **Format**: PNG preferred
- **Quality**: High resolution
- **Content**: Important features clearly visible
- **Naming**: Descriptive names (e.g., `login_screen.png`)
